# Assumptions

- The logistic map is used as the underlying chaotic system
- Finite precision arithmetic may affect the system's dynamics
- The method of converting real values to binary significantly impacts randomness
- NIST SP800-22 tests are used as a statistical benchmark, not as proof of cryptographic security 
- Initial conditions and parameters are assumed to form part of the cryptographic key space
- It is assumed that initial conditions and parameters can be securely shared or derived between communicating parties
